# Leantime Testing (Track B)

This repository contains the test assets, documentation, and CI configuration
for testing the Leantime open-source web application under Track B (Deploy & Test).

- System Under Test (SUT): Leantime (project management web app, deployed separately via Docker)
- This repo: test plan, test suites, assurance report, CI workflows, and evidence.

See `docs/local-setup.md` for instructions on running Leantime locally and executing tests.
